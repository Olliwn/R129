# Kärkkäinen Tool Shopping List

*This list comprises the tools required for the M119/722.3 maintenance, electrical work, and general R129 bulletproofing tasks.*

## Automotive Mechanics Tools
- [ ] **Large Jack Stands (Pukkisarja)** (Must comfortably hold a 2-ton vehicle securely)
- [ ] **Breaker Bar / Vääntövarsi** (1/2" drive, long handle for tough suspension/subframe bolts)
- [ ] **Torque Wrench / Momenttiavain** (e.g., 20-110Nm and 40-210Nm ranges for engine and wheel lugs)
- [ ] **Hex/Allen Socket Set / Kuusiokolohylsysarja** (Essential for Mercedes suspension and M119 components)
- [ ] **Torx Socket Set / Torx-hylsysarja** (Various sizes for interior and trim)
- [ ] **Oil Filter Wrench / Öljynsuodatinavain** (Sized for M119 filter housing cap)
- [ ] **Fluid Syringe / Nesteimuri** (For extracting ZH-M hydraulic fluid or power steering fluid)
- [ ] **Pry Bar Set / Sorkkarauta / Vääntörauta** (For testing control arms and engine mounts)
- [ ] **Transmission Funnel / Suppilo** (Long neck, narrow tip for 722.3 dipstick tube filling)
- [ ] **Magnetic Pick-up Tool / Magneettipoimija** (For dropped bolts in the M119 engine bay)

## Electrical & Prototyping Tools
- [ ] **Digital Multimeter / Yleismittari** (With continuity beeper and decent leads)
- [ ] **Wire Strippers / Kuorintapihdit**
- [ ] **Crimping Tool / Puristuspihdit** (For insulated/non-insulated automotive terminals)
- [ ] **Heat Shrink Tubing Assortment / Kutistesukkalajitelma**
- [ ] **Soldering Iron & Solder / Kolvi ja tina** (If decent models are available, maybe Gas powered e.g., Dremel Versatip for car use)
- [ ] **Automotive Test Light / Virtakynä** (12V)

## Consumables & Workshop Supplies
- [ ] **Brake Cleaner / Jarrukliineri** (Multiple cans - essential for cleaning old leaks)
- [ ] **Penetrating Oil / Irrotusöljy** (WD-40 or preferably CRC Rost Flash / MoS2)
- [ ] **Shop Towels / Teollisuuspyyhe** (Heavy-duty paper rolls)
- [ ] **Nitrile Gloves / Nitriilikäsineet** (Heavy duty)
- [ ] **Dielectric Grease / Sähkövaseliini** (For electrical connectors)
- [ ] **Threadlocker / Kierrelukite** (Medium strength / Blue)

*Note: Update this list based on specific fastener discoveries during the first inspection.*
